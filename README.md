# Digital-Design-With-Microcontrollers

This repository contains microcontroller-based projects ranging from displaying characters on a 4-digit 7-segment display to data communication via UART, information encryption, and local ring network communication.

Below is a list of the repository folders. Each folder is named according to the project it contains.
